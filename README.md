# terminal-app
Basic python terminal app with options menu, logo, colour prints, and progress bars

# Usage
Works for Windows, Mac and Linux
```
$ git clone https://github.com/almartDev/terminal-app.git
$ cd terminal-app
$ python3 main.py
```
